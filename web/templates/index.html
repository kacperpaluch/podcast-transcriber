Kolejka nowej transkrypcji

Wklej URL pliku audio — transkrypcja zostanie dodana do kolejki i przetworzona sekwencyjnie. Wyniki trafią webhookiem do n8n.

Integracja z n8n

n8n może automatycznie kolejkować transkrypcje przez API:

POST /api/transcribe
{
  "audio_url": "https://...",
  "language": "pl",
  "episode_title": "Tytuł odcinka",
  "feed_name": "Nazwa kanału",
  "guid": "opcjonalny-unikalny-id"
}
→ 202 Accepted  {"job_id": 42}
